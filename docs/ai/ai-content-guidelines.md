# AI Content Guidelines

AI-generated content inside the product must follow the same content standards as all other UI text.

## Rules

- generated text must remain concise and understandable
- generated explanations must stay task-focused
- generated content must not contradict system terminology
- generated text must avoid filler and promotional tone
- uncertain or low-confidence output should be phrased carefully or withheld when appropriate

## AI response structure

Preferred structure:

1. result
2. short explanation when useful
3. optional next action

The result should appear first.
