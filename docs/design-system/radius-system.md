# Radius system

Border radius refers to the rounding of the corners of an element's border in a design or layout. It is measured in units such as pixels or percentages and can range from completely square (0px) to completely circular (50% or 50px). By adjusting the border radius, designers can create a variety of shapes and styles, such as rounded rectangles, capsules, or circles. This property can be used to create a softer, more approachable aesthetic or to add emphasis and contrast to certain elements. The choice of border radius should take into account the intended use and context of the element, as well as the overall visual language and brand identity of the design system.

- Use default system from [Tailwind CSS](https://tailwindcss.com/docs/border-radius)

## Example

```tsx
<div className="rounded-lg">
```
