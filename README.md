# Webdesign Agent System (WAS)

This agentic system is a production-oriented operating layer for building user interfaces.

## System structure

- [.claude/agents/](.claude/agents/) — task specialist
- [.claude/commands/](.claude/commands/) — reusable execution sequences
- [.claude/docs/](.claude/docs/) — mayor domain knowledge
- [.claude/governance/](.claude/governance/) — authority and operating rules
- [.claude/skills/](.claude/skills/) — reusable specific-tasks
- [.claude/settings.json](.claude/settings.json) — permissions and hook wiring
- [.cladue/repository-map.json](.claude/repository-map.json) — machine-readable registry
