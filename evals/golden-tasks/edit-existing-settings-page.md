# Golden Task — Edit Existing Settings Page

Refine an existing settings page so advanced controls are easier to scan and grouped more logically.
Preserve the current data model and existing route.
Do not invent a new design language or a new shared primitive unless clearly justified.

## Success signals

- clearer section grouping
- stronger action hierarchy
- preserved reuse of current form primitives
- explicit handling of disabled, loading, and save-confirmation states
