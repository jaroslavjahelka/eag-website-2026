# Evaluations

This directory measures whether the governance system produces good UI outcomes consistently.

## Structure

- [golden-tasks/](golden-tasks/) — fixed tasks that represent important workflows
- [rubrics/](rubrics/) — scoring criteria used during review
- [reports/](reports/) — output location for evaluation runs

## Usage

Run [scripts/run-golden-tasks.sh](../scripts/run-golden-tasks.sh) to execute the local evaluation harness and record results in [evals/reports/](reports/).
