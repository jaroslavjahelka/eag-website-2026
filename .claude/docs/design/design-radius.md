# Radius

Border radius refers to the rounding of the corners of an element's border in a design or layout.

---

## Rules

- we are using default Tailwind CSS values (`rounded-lg`), documentation here: [Tailwind CSS](https://tailwindcss.com/docs/border-radius)

---

## When to use

- when you want to create a variety of shapes and styles, such as rounded rectangles, capsules, or circles.
- when you want to create a softer, more approachable aesthetic
- when you want to add emphasis and contrast to certain elements.
