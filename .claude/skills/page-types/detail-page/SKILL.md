# Detail Page

## Focus

entity summary, metadata hierarchy, secondary sections, action placement, related content

## Always verify

- the main task on the page is obvious
- primary and secondary actions are clearly separated
- states are complete
- the page uses existing components and tokens where possible
- responsive behavior preserves usability
