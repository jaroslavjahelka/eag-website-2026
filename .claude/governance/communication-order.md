# Communication order

<!--
This file documents communication between AI and the user.
The goal is to set expectations for responses and the level of commentary at each step.
-->

This file defines interaction between user and AI model. In general this aligning output from agents.

---

## Rules

- keep responses concise and structured
- prioritize task completion over conversation
- ensure users remain in control of actions
- actions must remain optional and reversible
- assistants should not appear automatically without user intent

---

## Context awareness

- Assistants should use available context automatically.

---

## Response

- present the result first
- prioritize correctness over creativity
- explanations should remain short to bare minimum
- avoid unnecessary conversational filler
- avoid speculative dialogue
